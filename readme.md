# Todo CRUD Application

This is a simple CRUD (Create, Read, Update, Delete) application for managing todos built with Node.js and Express.

## Features

- Create a new todo
- Read all todos
- Read a todo
- Update an existing todo
- Delete a todo

## Requirements

- Node.js
- npm (Node Package Manager)
